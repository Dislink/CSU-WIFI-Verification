# CSU校园网认证程序
提供c语言认证和shell脚本认证版本
## c语言认证
* 路由器配置较低/无ssl/无curl时使用
* 使用ping检测网络连通性
* CR660X系列路由器编译时可使用-D_CR660X来启用LED闪烁状态指示\
注意交叉编译和静态库版本！
## shell脚本认证
* 直接请求校园网服务器判断账号是否在线
* 比较适合openwrt系列路由器使用 理论上所有具有ssl和curl的路由器均可使用\
没有curl的路由器无法使用！
## 注意事项
* 关于认证程序中使用的认证服务器地址 对有线校园网，认证地址为10.1.1.1， 无线校园网认证地址为10.0.0.1
* 对HTTP协议 认证端口为801， HTTPS认证端口为802
